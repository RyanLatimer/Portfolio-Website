StockSight: Stock Price Prediction and Analysis

StockSight is a stock price prediction project using TensorFlow machine learning algorithms. It analyzes historical stock data and financial data to make future predictions./p>

StockSight uses a neural network built on the TensorFlow library to discover hidden nonlinear patterns across thousands of data points

StockSight will be deployed as a free, open source application to provide financial analysis to everyone.
